Restaurant Web Application Project

Sushi Cafe

Languages & Platforms used: Python-Flask, MySQL, Javascript, JQuery, HTML, CSS, Tableau

ABOUT THE PROJECT

The objective of project is to build a python flask application for restaurants. After finishing the order, customer will be directed to the survey on which the marketing analysis is based. As an owner, you can not only monitor the operation, but can view thorough analysis via Tableau. In addition, owner could utilize Apriori Algorithm to identify which items are most frequently bought together, which help create 'combo' manu.

The data source is from people who order food from website. We collected data from friends and coworkers.
