Leaflet.js Project

Mapping Earthquate

Languages & Platforms used: Javascript, Leaflet.js

ABOUT THE PROJECT

The objective of project is to provide the visualization that map out earquake data on a real time basis. The data on the map is presenting the collection of earthquake data across the world for previous 7 days. The circle sizes as well as colors commensurate with the magnitude of earthquake.

The data is based on GeoJson from USGS, which provides earthquake data in a number of different formats, updated every 5 minutes. For more details, please visit : https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php
